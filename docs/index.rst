Gaia Twitter Plugin
================================

This is a plugin for Gaia (https://github.com/OpenDataAnalytics/gaia) that
loads twitter streams into geojson


Installation
-----------------

- git clone https://github.com/OpenDataAnalytics/gaia-twitter-plugin.git
- cd gaia-spatialstats-plugin
- pip install -e .
- pip install -r requirements


Testing
-----------------

- python -m unittest discover
